## 正文

OpenRouter 上有一个叫 Owl Alpha 的匿名模型。过去两个月，它在 Hermes Agent Workspace 调用量排第一，Claude Code 工作流排第二，日峰值 559 亿 token，累计处理 10.1 万亿 token。没有品牌曝光，没有融资发布会，没有创始人站台。就是一个匿名 API，在 Agent 开发者的工作流里安静地跑着。

6 月 30 日，美团揭开了面纱：Owl Alpha 就是 LongCat-2.0。1.6 万亿参数 MoE，原生 100 万 token 上下文，MIT 开源。SWE-bench Pro 59.5——超过 GPT-5.5 的 58.6。

多数人的第一反应是"美团也做大模型了"。这个理解太浅。真正值得看的是：**一个模型在完全匿名的情况下，仅靠 Agent 任务完成率拿了两个月第一。** 身份揭开后，月均调用量还在涨。

这件事暴露了一个被忽略的市场结构差异。消费者 AI 靠品牌驱动——用户用 ChatGPT 因为知道 OpenAI。Agent API 市场靠结果驱动——开发者选工具看的是这个 API 在工作流里的实际输出。在后者的市场里，你可以匿名两个月排第一，因为唯一的信任信号是任务完成率，不是名字。

这也解释了美团为什么选择匿名入场。先用结果建立真实占有率，再用身份加固。如果一开始官宣"美团做了 Agent 编程模型"，开发者的第一反应多半是"外卖公司做什么 AI"——品牌认知反而是阻力。匿名让产品绕过了身份偏见，直接在结果层面竞争。

还有一个容易被忽略的变量：**5 万张国产 ASIC，35 万亿 token 预训练，从预训练到推理全流程不用 NVIDIA。** DeepSeek V4 Pro 推理用国产芯片，但训练仍依赖 NVIDIA。LongCat-2.0 是目前公开披露的第一个训练和推理全流程国产算力完成、且有第三方可验证性能的万亿参数模型——可验证性不靠白皮书，靠 OpenRouter 上两个月的真实调用量。

发布促销价 $0.30/M input，$1.20/M output。Claude Opus 4.8 的大约十分之一。

**Agent API 市场的竞争逻辑已经和消费者 AI 分岔了。品牌、融资规模、创始人光环——在这个市场里全是噪音。唯一的硬信号是任务完成率。** 美团证明了一件事：在一个纯粹以结果说话的市场里，外卖公司的模型可以匿名击败 GPT-5.5。
