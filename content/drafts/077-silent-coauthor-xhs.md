## 正文

让 Anthropic 从深度亏损走向季度盈利的，不是更聪明的聊天机器人。

SemiAnalysis 近日发布的 Anthropic IPO 财务分析报告给了一组和直觉相反的数字。Anthropic 的 ARR 从 2025 年底的 90 亿美元，在 6 个月内增至 600 亿美元——7 倍。整体毛利率从 2024 年的 -94% 翻到 +60%。但消费者订阅——也就是大多数人最熟悉的"和 Claude 聊天"——只占 ARR 的约 5%。75%-85% 的收入来自 API 计费。API 最大的消耗者是 Claude Code。不是对话。是代码。

Claude Code 的规模可以用一个数字直接感受：它已经贡献了全球 GitHub 提交量的 7% 以上。每 14 次 commit，有 1 次是它写的。这些代码不是草稿——通过了 code review，通过了 CI 测试，被合并进了主分支。和旁边 13 条人类写的 commit 放在一起，你在 git log 里分不出来。

代码库的基因正在被静默改写。

Claude Code 创建者 Boris Cherny 在 Sequoia AI Ascent 上说过一句话：他 2026 年没有手写过一行代码。每天从手机上合并 50 到 150 个 PR，全部来自 AI agent。他的原话是 coding is solved for me。但后半句更重要——真正的 gap 不在技术，在组织。不是 AI 不能写代码，是团队还不知道怎么在新的分工下工作。执行层已经被接管。判断层还是人的。中间没有过渡期。

SemiAnalysis 这份报告量出来的不只是一家公司的利润表。它标定了一个阈值：当一个 AI 工具贡献了全球 7% 的代码，它已经不是"辅助工具"——它是软件生产的结构性参与者。

**AI 协作工具不是锦上添花的产品功能——它是 AI 公司盈利模式的地基。从 -94% 到 +60% 的毛利率翻转已经证明了这一点。没有人宣布这个转变。没有人投票。它只是出现在了全球 7% 的 commit 记录里。**
