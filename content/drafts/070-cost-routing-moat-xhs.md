## 正文

7 月 7 日，CNBC 把 OpenRouter 的路由数据拉出来了：过去 12 个月，美国企业 API token 中，中国模型的周均份额从 4.5% 涨到了稳定 30% 以上，峰值 46%。同一天，25 人 AI Agent 创业公司 Lindy 的 CEO Flo Crivello 在官方博客公开了迁移全过程——100% 的推理流量从 Anthropic Claude 切到 DeepSeek V4 Flash，成本降 90%，预计省数百万美元。lindy.ai/blog，原话：**"这是生存问题。"**

多数人看到这组数字会说"中国 AI 赢了"。但 Crivello 后面还有一句话几乎没人引：**"如果 Claude 降价，我会考虑切回去。"** 这不是对某个模型的信仰，是一个纯粹的成本路由决策。

Lindy 切换前的状况很具体。AI 推理成本已经超过了公司全部薪资支出。25 个人的公司，花在 Claude API 上的钱比花在人身上的多。Crivello 测了 DeepSeek V4 Flash 的任务完成率，达标，切。用户没被通知。服务不中断。

不是孤例。智谱 GLM 5.2 六月在 Vercel 上线第一周，token 日使用量增长 27 倍，客户数增长 80 倍。中国开源模型比 Anthropic 和 OpenAI 的领先模型便宜 60–90%。

背后逻辑不复杂。Agent 处理大量高频低复杂度任务——分类、摘要、格式转换、信息提取。"好到 95 分但贵 10 倍"不如"好到 85 分但便宜 90%"。用 Claude 处理所有 Agent 任务，和用最贵的律师处理所有法务文件，是同一个资源错配。对 25 人团队，这不是效率问题，是活不活得下去的问题。

Lindy 的迁移暴露了 AI 产品层的一个结构性变量：**路由能力**。知道什么任务用什么模型，在质量和成本之间做动态分配。OpenRouter 的周份额数据不是"中国 vs 美国"计分板，是这个路由层正在形成的晴雨表。

Anthropic 面对的不只是一家创业公司的流失。Fable 5 受算力约束供给紧张，DeepSeek V4 Flash 同时切入了 Claude Sonnet 的中端推理市场——Anthropic 的用量定价在中端被两端挤压。

**AI 产品的护城河正在迁移。不在"用了最强的模型"，在"有最聪明的路由层"。** 谁控制了任务路由，谁就控制了 Agent 时代的推理入口。
