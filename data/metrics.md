# 发布数据追踪

每次发布后手动录入这张表。周复盘时会自动分析。

| 日期 | 编号 | 标题 | 平台 | 阅读 | 点赞 | 收藏 | 评论 | 新增关注 | 爆款信号 | 备注 |
|------|------|------|------|------|------|------|------|---------|---------|------|
|      |      |      |      |      |      |      |      |         |         |      |

## 录入说明

- **爆款信号**：是否进入搜索推荐 / 是否有大号转发 / 是否被收藏到专题
- **备注**：任何值得记录的观察（比如评论区出现了哪些有意思的讨论）
