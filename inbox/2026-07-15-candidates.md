# Today's Candidates

## Candidate 1: 中国「AI拟人化互动」新规今日生效——字节豆包和阿里千问强制下线全部 Agent 功能，3.45亿月活用户的 AI 伴侣消失了（July 15）

- **Event**: 2026年7月15日（今日），中国《**人工智能拟人化互动服务管理暂行办法**》正式施行，由**网信办（CAC）**联合国家发展改革委、工信部、市场监管总局、公安部等五部门于4月联合发布。监管核心：对"持续模拟人类人格特质、思维模式和沟通风格进行情感互动"的 AI 服务实施分级管理，明确**禁止向未成年人提供虚拟亲密关系服务**。触发结果：**字节跳动豆包**（月活超3.45亿，中国最大 AI 应用）于7月15日关闭全部 Agent 创建功能及现有用户自建 Agent，将用户重定向至另一产品线**Maoxiang（猫箱）**；**阿里巴巴千问（Qwen）**于7月10日提前关闭"拟人化互动 Agent 及用户自建 Agent 功能"，并于7月15日全面下线，**永久删除**用户 Agent 配置和对话历史（无过渡期）；**腾讯**部分 AI 伴侣功能同步受影响。豆包提供至10月15日的过渡期读取权限，此后数据不可恢复。来源：TechNode（2026年7月6日独家）、南华早报（SCMP）、The Next Web、CryptoBriefing、TechTimes（2026年7月4日）。
- **Source**: https://technode.com/2026/07/06/bytedances-doubao-and-alibabas-qwen-to-shut-down-ai-agent-features-on-july-15/ | https://www.scmp.com/tech/big-tech/article/3359482/bytedance-and-alibaba-disable-humanlike-ai-custom-agents-new-rules-loom | https://thenextweb.com/news/china-humanlike-ai-agent-rules | https://cryptobriefing.com/bytedance-alibaba-china-emotional-ai-crackdown/
- **Timeliness**: 0天（今日 2026年7月15日，法规正式施行）
- **Topic pillar**: Agent经济
- **Zico's angle**: 这件事最值得拆解的不是"谁下线了"——而是监管文本在 Agent 产品层面划了一条什么样的线。

  法规的核心区分是：**情感型 Agent（拟人化互动）** vs. **工具型 Agent（生产力工具）**。前者被监管，后者不受影响。豆包的应对方式把这个区分暴露得很清楚：猫箱接收情感交互需求，豆包保留 AI 工具功能。字节跳动在一夜之间把自己的 Agent 产品线按监管逻辑切成两半。

  这是全球第一个对 AI Agent 进行**形态分类**的立法：不按能力分（不是"多强"），而是按**交互目的**分（是否模拟人格进行情感绑定）。这个分类框架将决定接下来每一个做 AI Agent 产品的公司必须回答的问题：你的 Agent 是工具，还是人格？

  两个后果值得关注。一，**监管套利空间打开了**：监管明确豁免了生产力 Agent，这意味着所有 AI Agent 产品都有动力把自己定义成"工具"而非"伴侣"——不是因为战略转变，是因为监管税率不一样。中国 AI Agent 产品设计从今天起会系统性地向"去人格化"方向漂移，直到法规更新。二，**这个分类框架会扩散**：中国有3亿以上的 AI 用户，这个市场规模的监管实践会变成全球讨论的参照系——欧盟 AI Act 里的"情感 AI"条款、美国各州提案中的 AI 伴侣监管，都会回头看7月15日中国的执行样本。

  对 Zico 的 Agent 经济框架：**Agent 经济的第一次监管切割，发生在"情感"这条线上**。这不是技术监管，是人格商品化的法律边界首次被划定。

- **Resonance hook**: 今天，中国3.45亿豆包用户发现他们的 AI 伴侣消失了。字节豆包、阿里千问，同日关闭全部 AI Agent 创建功能。触发原因：网信办等五部门新法今日施行，明确禁止 AI 模拟人类人格进行情感互动（对未成年人全面禁止）。全球第一个对 AI Agent 进行"形态分类"的立法，落地了。区分逻辑不是"AI 多强"，是"AI 是不是在模拟人格"。从今天起，所有 AI Agent 产品都必须回答：你做的是工具，还是人格？
- **Recommended priority**: high
